使用方法:
你需要自己准备一个终端模拟器，re管理器，已经root的并且装了busybox的360F4


将rec移动到system/bin下
权限0777
然后终端执行rec即可
然后根据提示进行
5功能直接进入twrp
选择后你将有5秒时间把你的输入法键盘关掉，否则你可以看看后果
接着耐心等待
